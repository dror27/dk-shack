# עֵץ הַלִּימוֹן

עֵץ הַלִּימוֹן עֵדִּי\
בִּפְרִיחָתוֹ הַשִּׁשִּׁית \
כִּי כָּל הֶחָמוּץ \
מְקוֹרוֹ רֵיחָנִי\
תּוֹכוֹ רָטֹב \
וּקְלִפָּתוֹ בַּת סוֹד\
\
עֵץ הַלִּימוֹן עֵדִּי\
מְקוֹרוֹ רֵיחָנִי\
\
עֵץ הַלִּימוֹן עֵדִּי\
כִּי גַּם אֲנִי, \
בַּעֲשׂוֹרִי הַשִּׁשִּׁי\
חָמוּץ רָטֹב וְרֵיחָנִי\
קְלִפָּתִי נֶשֶׁק \
נֶשֶׁק סוֹדִי\
\
נְשִׁיקָתִי לִימוֹן\
לְמּוֹן-שֶׁרִי\
