# אנאלי בנאלי

אָנָאלִי, בָּנָלִי, דַּבֵּר אִתִּי בְּלִחְשׁוּשִׁים\
טוֹטָאלִי, וֶרְבָּלִי, אֶל תַּעֲשֶׂה לִי רִגּוּשִׁים\
בַּשֶּׁקֶט, לֹא רַע לִי, \
תָּבוֹא בַּלַּיְלָה, אֶל בֵּין הַסְּדִינִים\
\
וּבַדִּמְיוֹן, אֲנִי רוֹקֶדֶת עִם חַיּוֹת\
עָלִים גּוֹרֶפֶת, מַאֲכִילָה פָּרוֹת\
בָּאָחוּ, עַל גִּבְעָה, מֵעַל גַּגּוֹת הָעִיר\
אִישׁ לֹא מַגִּיעַ, אוֹתִי מֵעִיר\
\
וְאַתָּה דִּיגִיטָלִי, מָלֵא בְּחִדּוּשִׁים\
מַקְפִּיץ לִי בָּנָנָה, בְּרֹטֶב שֶׁל עִזִּים\
מַשְׁקִיעַ, מַגִּיעַ, וְאָז יוֹרֵד לִי מֵהַפַּסִּים\
הַכֹּל פֹּה רָגוּעַ, צֵא בַּחוּץ, תָּבִיא עֵצִים\
\
אָנָאלִי, בָּנָלִי …\
\
עֲנָנִים עָבְרוּ, שֶׁמֶשׁ נִרְדְּמָה\
בְּמֶרְחַק הֵדִים, בְּרָקִים שֶׁל מִלְחָמָה\
הֶעֱמַדְתִּי תֶּה, יֵשׁ מַיִם חַמִּים\
בּוֹא נִתְכַּרְבֵּל, קֹדֶם כֹּל יְשֵׁנִים\
\
אָנָאלִי, בָּנָלִי …\
\
טוֹב, אָז טִפַּסְתָּ שָׁעָה\
אֶל הַנְּסִיכָה הַנְּצוּרָה\
אֲנִי כְּבָר פֹּה שְׁלוֹשִׁים שָׁנָה\
\
וְאַתָּה … אַתָּה אָנָאלִי, בָּנָלִי, דַּבֵּר אִתִּי בַּלִּחְשׁוּשִׁים
